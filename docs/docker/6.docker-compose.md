# docker-compose

### 配置文件

```yaml
version: '3'
services:
  redis:
    container_name: redis
    image: redis
    command: redis-server /usr/local/etc/redis/redis.conf
    ports:
      - 6379:6379
    volumes:
      - /jjtest/redis/data:/data
  nginx:
    container_name: nginx
    image: nginx
    privileged: true
    ports:
      - 80:80
    volumes:
      - /jjtest/nginx/html:/usr/share/nginx/html
```
