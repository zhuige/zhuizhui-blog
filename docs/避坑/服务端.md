# 服务端

## session 实现登录

koa-session

使用 session 实现登录记得要前端设置可以跨域请求携带 cookie 比如 axios

```javascript
const req = axios.create({
  baseURL: 'http://localhost:3000',
  withCredentials: true,
})
```

并且服务的的跨域也要设置 例如 使用的是 koa-cros

```javascript
app.use(
  cors({
    credentials: true,
  })
)
```

不然就会出问题
